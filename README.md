# ORA-Style Research Office Website

A 5-page responsive website built from scratch to simulate a university 
research administration office, modeled after the UMD Office of Research 
Administration (ORA).

## Pages
- **Home** — Hero section with office overview and quick links
- **Announcements** — Latest funding deadlines and policy updates
- **Staff Directory** — Team members with contact information
- **Resources** — Forms, templates, and compliance checklists
- **Contact** — Inquiry form for faculty and staff

## Features
- 100% responsive across 3 breakpoints (desktop, tablet, mobile)
- Built with zero external CSS frameworks — pure HTML5 and CSS
- CSS Flexbox for layout and media queries for responsiveness
- Semantic HTML5 structure mirroring CMS template architecture
- Modular CSS designed for easy Craft CMS migration

## Tech Stack
HTML5, CSS3, JavaScript

## How to Run
1. Clone the repo
2. Open `index.html` in any browser
   Or use VS Code Live Server for live reload
